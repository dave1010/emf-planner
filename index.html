Favourites

You need to manually re-import your favourites whenever you change them, but it only takes a second:

  1. Make sure you can see your favourites on the EMF site.
  2. Long press or right click on this link.
  3. Select Download link or Save Link As...
  4. Upload the downloaded favourites.json file below.
